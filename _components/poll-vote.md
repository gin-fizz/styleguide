Once a poll is created, you can vote on it like this:

```html
<div class="poll--vote">
  <label for="optionXX">Some option :)</label>
  <input type="checkbox" name="optionXX" id="optionXX">
</div>

<div class="poll--vote">
  <label for="optionXX">Some option :)</label>
  <input type="radio" name="optionXX" id="optionXX">
</div>
```
